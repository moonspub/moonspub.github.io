Dealing with sprays

We have some commands and tools for dealing with sprays. This post goes over how to use them.

spraybanning

To sprayban, you can use the sm_sprayban command. sm_sprayunban does the inverse.

doing a spray check

If you want to check the sprays of each player you can do so using the sm_adminspray command. It allows you to spray other players' sprays where you're looking to view them. You should do so in a spot outside of the map where people are unlikely to look. This command works in spectator mode so you can use that to noclip to a spot out of view.

identifying and removing sprays

You can identify a spray using the sm_spraytrace command. This will tell you who sprayed it. To remove the spray, use the sm_qremovespray command. With the Q at the beginning.

It's worth mentioning that while some players may have NSFW sprays they may not intend to spray them. Unless the spray itself communicates a problem with the player (i.e. you see some Nazi shit) then you shouldn't take action if it's unlikely they sprayed it.

if the plugin doesn't work

If for some reason the plugin doesn't work or you can't see the spray, you have the backup option of using the server logs.

In the logs, it will log on the first occurrence of a user using a certain spray. The message looks like this:

Spray: Skinny Soviet (STEAM_0:1:41787730) -> <Spray: 269008e4>

The set of characters in the spray part is the "hex" of the spray. If you were on the server when the file was sprayed there should be a file with the hex suffixed with .dat (so in the above example 269008e4.dat) in your tf2classified/download/user_custom folder. The file will exist in a folder that's the first two characters of the hex. So in this case user_custom/26/269008e4.dat. Despite the extension being .dat it's indeed a VTF file.

If it isn't there, you can ask Moonrder to look in the server's files for it.

You can use a program like MareTF for Windows/Linux, VTF Viewer for Windows, the no_vtf CLI, or one of the plethora of tools that can view the VTF file.
