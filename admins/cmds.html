Admin commands

Player actions

  • sm_kick <user> <reason> - Kick a player.
  • sm_ban <user> <time> <reason> - Ban a player (time in seconds, 0 = indefinite).
  • sm_unban <steamid or ip> - Remove a ban.
  • sm_gag <user> - Temporarily prevent text chat.
  • sm_mute <user> - Temporarily prevent voice chat.
  • sm_silence <user> - Temporarily prevent both text and voice chat.
  • sm_unsilence <user> - Remove any silence, gag, or mute.
  • sm_rename <user> - Change a player's username.
  • sm_slay <player> - Kill the player.
  • sm_zensitive_never <steam id> - Prevent a user from consenting to sensitive messages and revoke existing consent.
  • sm_zensitive_reset <steam id> - Reset sensitive chat settings and remove the restriction applied by sm_zensitive_never.
  • sm_nag <user> <nagpage> - Shows a premade message to the player via their HTML MOTD. Has a brief text fallback. Current list of valid nags: dj, micspam, nsfw, toxic, test

Sprays

  • sm_spraytrace - Trace the spray you're looking at.
  • sm_removespray - Remove the spray you're looking at and display the moderation menu.
  • sm_qremovespray - Remove the spray you're looking at without displaying a menu.
  • sm_adminspray <user> - Force another user's spray where you're aiming.
  • sm_sb <user> - Spray ban a player.

Utility

  • sm_csay <message> - Display a message at the top-centre of the screen.
  • sm_uid <user> - Get one or more players' Steam IDs in standard format.
  • sm_uid64 <target> - Get a player's 64-bit (community) Steam ID.
  • sm_steamhistory/sm_history <user> - Get the steamhistory.net URL for one or more players.
  • sm_setnextmap <map> - Change the next map in the current rotation.
  • sm_settings - Change chat tag settings.
  • sm_opensteamhistory / sm_openhistory <target> - Open the player's steamhistory.net page in an in-game MOTD panel.

DJ stick

  • sm_djstick_force / sm_forcedj <target> - Force a player to become DJ regardless of whitelist status.
  • sm_djstick_revoke / sm_revokedj <target> - Force the current DJ to abandon DJ status.

Fun

  • sm_vote <question> <answer1> <answer2>... - Create a vote with the supplied question and answers.
  • sm_smash <user> - Smash player with a hammer.
  • sm_smite <user> - Smite the player.

Class Consciousness

  • sm_cc_roll - Manually re-roll the class matchup.
  • sm_cc_change <red/blue/green/yellow/all> <class> - Change the class matchup for one or all teams.
  • sm_cc_enable <0/1> - Toggle class enforcement.

Senior Admins Only

  • sm_addban 0 <steamid> <reason> - Add a permanent ban to the SourceBans database.
  • sm_trigger_serverexplosion - Explode the server (Senior Admins only).
  • sm_rcon <cmd> - Execute an RCON command.
  • sm_condify <target> <condition number> - Apply a condition to a player.
  • sm_decondify <target> <condition number> - Remove a condition from a player.
  • sm_classify <target> <class> - Change a player's class without changing weapons.
  • sm_reclassify <target> <class> - Change a player's class and refresh their weapons.
  • sm_regenify <target> - Regenerate a player as if they touched a resupply cabinet.
  • sm_respawnify <target> - Immediately respawn a player.
  • sm_teamify <target> <team> - Move a player to another team, killing them in the process.
  • sm_moneyify <target> <amount> - Set a player's MvM balance to the specified amount.
